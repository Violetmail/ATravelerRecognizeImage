v1.0
这是一个基于图片匹配的自动操作脚本，可以实现部分软件操作的自动化。

我根据gooey库，制作了一个图形化界面，用来调用参数和用户选项。

使用方法：

在cmd.xlsx文件中配置每一步的指令，根据表格提示填入对应的内容，指令1234567，分别具备单击，双击，右键单击，输入，等待，滚动，判断的功能。

必要的，对于判断功能，为判断表格中的额外图片是否被匹配到，匹配到则单击它，否则与指令“1”功能类似。
